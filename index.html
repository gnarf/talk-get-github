So what is GitHub?

GitHub is a collaborative code managment tool. Its mascot is the "Octocat" pictured here. It builds on top of git.

First, What is Git?

  • Revision Control
  • Distributed
  • Speed
  • Data Integrity
  • Non-Linear Workflows
  • Most Popular (source)
Poll: How many people have never used git? have you used other source control? In case you aren't familiar, git is...
→ code goes in a git repository... A history of the codebase, revision, control
→ distributed means every "clone" is it's own local repo (offline anyone)
→ speedy because it's local for most operations
→ because sha1 checksums
→ non-linear... branching and mergeing are super fast, A core assumption in Git is that a change will be merged more often than it is written
→ popular! yay
