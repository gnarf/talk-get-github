Okay! So what is GitHub?

Build software better, together.
(Emphasis mine)

Another quick poll - how many of you already have a GitHub account? Do you use it for work? For open source?
For those of you that don't know - GitHub - aside from having an adorably cute mascot - is a place for hosting and collaborating on git repositories. Their Motto ‚Üí really kind of sums it up.

Together...

  • Sharing Knowledge
  • With history for reference
  • Asynchronusly
I want to zero in on the word "together" for a second. What means to me is sharing our knowledge and efforts in an online space, with history for reference, and asynchrounsly. This means you aren't tapping your co-worker on the shoulder and demanding they give you time, and we all get to be more productive.

GitHub

(in case you aren't already convinced)
  • Integrated Issue Tracking
  • Collaborative Code Review (Pull Request)
  • Organizations & Teams
  • Robust API
  • Integrations
  • Free for public
  • Pay for private
  • Pay more for Enterprise
For those of you not using GitHub at work, here's a few selling points in case you want to convice your team.
Building on top of git's powerful tools for managing your code, GitHub adds a bunch of great features.
‚Üí They have an Integrated Issue Tracking system which is something every project needs to solve. You can of course disable it if you already track your issues elsewhere.
‚Üí Collaborative Code Review! .... Pull Requests A.K.A. "PR"s are probably the most important feature offered by GitHub and I'll tell you all about em in a minute or two.
‚Üí github is also Team oriented. you can create organizations, group them by teams, and control access to repos individually.
‚Üí GitHub's website is built using their own API. Almost anything you can do on GitHub can be done with the very well documented API.
‚Üí And because the API is awesome, it integrates with a lot of services. Flowdock, HipChat, Slack, IRC, Jenkins, Travis, JIRA, the list goes on and on.
‚Üí It's free for public repositories, even organizations! If you aren't trying to hide it from the world, you'll never pay a dime. If you pay you unlock private repositories.
‚Üí If you pay even more you can get Enterprise, and run it on your own servers.
